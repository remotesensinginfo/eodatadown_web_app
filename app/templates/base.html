Background

We write a number of tools which aid the processing remotely sensed and spatial data. Primarily the Remote Sensing and GIS software library (RSGISLib) which under pins most of our data analysis projects and software such as ARCSI and EODataDown which combined to provide a fully automated processing chain for downloading and converting remotely sensed data, such as Landsat, Sentinel-1 and Sentinel-2 to an Analysis Ready Data (ARD) format.
