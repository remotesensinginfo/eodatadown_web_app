Open in Desktop GIS

To use within a desktop GIS two options are available, the first is to get the original ARD GeoTIFF files from the server, see system path above, but these files are very large (Landsat 500Mb, Sentinel-1: 9Gb Sentinel-2: 2Gb) when including all the files generated for the ARD product. Alternatively, you can download a pre-rendered MBTiles file (used for the map above) which is significantly smaller (Landsat 50Mb, Sentinel-1: 200Mb Sentinel-2: 50Mb) and can be opened in your Desktop GIS (e.g., QGIS) alongside your other datasets for visualisation. MBTiles can therefore also be used offline (i.e., without an internet connection).
